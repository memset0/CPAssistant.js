## Features

### 在 Vjudge 上 Fork 来自其他 OJ 的比赛

当您打开其他 OJ（当前支持：CodeForces）的竞赛场次界面时，脚本会创建一个 `Fork Contest (VJ)` 按钮，点击后将自动跳转到 Vjudge 并创建一场题目内容相同的虚拟比赛。
